# Dual Ace: Sky Rift

Unreal Engine C++ 기반 3D 플라이트 슈팅 게임 프로젝트입니다.  
중간고사 블루프린트 프로젝트의 핵심 아이디어인 1인칭/3인칭 전환, 캐릭터 선택, wave 전투 흐름을 유지하되, 기말 조건에 맞춰 주요 게임 로직을 C++ 중심으로 구현했습니다.

## 프로젝트 목표

- C++만으로 플레이어, 적, 보스, wave, UI 흐름, 리더보드, 사운드 재생 규칙 구현
- Lobby -> Character Select -> Gameplay -> Result 구조 제공
- Falcon / Titan 2개 플레이어 기체 선택
- 1인칭 / 3인칭 시점 전환
- wave가 진행될수록 어려워지는 플라이트 슈팅 전투
- 보스 wave와 보스 HP UI
- 게임 종료 후 리더보드 표시
- 데이터 테이블로 플레이어, 적, 보스, wave 밸런스 조정
- 직접 제작한 기본 도형 기반 플레이어/적/보스/맵 에셋 사용

## 화면 흐름

```text
Lobby
  - Game Play: 캐릭터 선택 화면으로 이동
  - Dashboard: 리더보드 확인
  - Exit: 게임 종료

Character Select
  - Falcon 선택
  - Titan 선택
  - Launch: 선택한 기체로 게임 시작
  - Back: Lobby로 복귀

Gameplay
  - 무한 맵 형태의 비행 전투
  - wave 단위 적 스폰
  - 일반 적, 빠른 적, 탱커 적, 보스 등장
  - 플레이어 HP, 적 HP, 보스 HP, 점수, wave 진행 표시

Result
  - 최종 점수 표시
  - 생존 시간 / 도달 wave 저장
  - 리더보드 Top 5 표시
```

## 조작법

| 입력 | 기능 |
| --- | --- |
| `W` | 앞으로 이동 |
| `S` | 뒤로 이동 |
| `A` | 왼쪽 이동 |
| `D` | 오른쪽 이동 |
| `Q` | 왼쪽으로 기울기 |
| `E` | 오른쪽으로 기울기 |
| `Left Mouse Button` | 기본 사격 |
| `V` | 1인칭 / 3인칭 시점 전환 |
| `R` | 궁극기 미사일 발사 |
| `Esc` | 일시정지 / 볼륨 조절 UI |
| `1` | Falcon 선택 |
| `2` | Titan 선택 |
| `Enter` | 선택 확정 / 게임 시작 |

`A/D`는 좌우 이동만 수행하며 기체 회전은 주지 않습니다.  
`Q/E`는 기체 기울기 연출만 담당하며, 키를 누른 순간 기울기 음성이 1번만 재생됩니다.

## 플레이어 기체

### Falcon

빠른 이동과 빠른 연사가 장점인 경량 기체입니다.

- 낮은 HP
- 빠른 이동 속도
- 빠른 사격 주기
- 회피 중심 플레이에 적합

### Titan

높은 체력과 강한 공격력이 장점인 중장갑 기체입니다.

- 높은 HP
- 느린 이동 속도
- 강한 탄환 피해
- 안정적인 전투에 적합

## 전투 시스템

- 플레이어, 일반 적, 보스 모두 HP를 가집니다.
- 적은 플레이어와 같은 높이에서 추적하며 충돌 공격을 합니다.
- 적과 부딪히면 플레이어가 피해를 받고 `아프다` 음성이 재생됩니다.
- 벽 또는 맵 경계와 부딪히면 플레이어가 피해를 받고 `벽에 부딪힘` 음성이 재생됩니다.
- 플레이어 HP가 10% 이하가 되면 `딸피 딸피` 음성이 1회 재생됩니다.
- 플레이어가 사망하면 `이걸 죽네` 음성이 재생되고 게임이 종료됩니다.

## Wave 시스템

wave마다 적 스폰 조건과 난이도가 달라집니다.

- wave별 스폰 간격
- 한 번에 스폰되는 적 수
- 기본 적 속도 / HP / 피해량
- 빠른 적 등장 확률
- 탱커 적 등장 확률
- 보스 wave 여부

wave가 바뀌면 wave 안의 킬 카운트는 다시 0부터 시작합니다.  
보스 wave에서는 일반 적을 정리하고 보스가 등장합니다.

## 킬콜 사운드 규칙

wave 안에서 킬 순서에 따라 음성이 재생됩니다.

| wave 내 킬 수 | 재생 음성 |
| --- | --- |
| 1킬 | 퍼스트 킬 |
| 2킬 | 더블킬 |
| 3킬 | 트리플킬 |
| 4킬 | 쿼드라킬 |
| 5킬 | 펜타킬 |
| 6킬 | 사용자가 미쳐 날뛰고 있습니다 |
| 7킬 이상 | 잡았쥬 |

킬콜은 피격, 벽 충돌, 시점 전환, 기울기 음성과 동시에 들릴 수 있습니다.  
예를 들어 적을 잡는 동시에 맞으면 `퍼스트 킬`과 `아프다`가 같이 들릴 수 있습니다.

## 궁극기 시스템

`R` 키를 누르면 큰 대포 미사일 형태의 궁극기 발사체가 발사됩니다.

궁극기 충전 규칙:

- wave 안에서 7킬을 하면 궁극기가 활성화됩니다.
- 궁극기 활성화 시 `궁 준비 완료` 음성이 재생됩니다.
- wave `n`에서 궁을 사용했다면 wave `n + 1`에서 다시 7킬을 해야 재활성화됩니다.
- wave `n`에서 궁을 사용하지 않았다면 wave `n + 1`에서도 궁 사용 가능 상태가 유지됩니다.
- 궁으로 적을 처치한 것도 wave 킬 카운트에 포함됩니다.
- 예를 들어 6킬 상태에서 궁으로 적을 처치하면 7킬로 집계되어 바로 궁 재충전 조건을 만족할 수 있습니다.

궁극기 사운드 우선순위:

- `궁 준비 완료`, `궁 발사`는 중요한 음성으로 처리됩니다.
- 이미 재생 중인 소리는 끊지 않습니다.
- 현재 소리가 끝나면 대기 중인 일반 음성보다 `궁 준비 완료` 또는 `궁 발사`가 먼저 재생됩니다.

## 사운드 시스템

`Content/Audio`의 음성 파일을 Unreal SoundWave 에셋으로 import하여 C++에서 로드합니다.

적용된 주요 사운드:

- UI 클릭: `클릭`
- 캐릭터 선택 진입: `캐릭터를 선택 하시오`
- 게임 시작: `게임을 시작하지`
- 1인칭 전환: `1인칭 전환`
- 3인칭 복귀: `3인칭 복귀`
- 기울기: `기울기`
- 궁 발사: `궁 발사`
- 궁 준비 완료: `궁 준비 완료`
- 적 피격: `아프다`
- 벽 충돌: `벽에 부딪힘`
- 저체력: `딸피 딸피`
- 플레이어 사망: `이걸 죽네`
- 보스 처치: `보스 컷`
- 보스 클리어: `보스 클리어`
- 게임 종료 버튼: `사용자가 게임을 탈주하였습니다`

사운드는 목적에 따라 두 개의 큐로 분리되어 있습니다.

- 킬콜 큐: 퍼스트킬, 더블킬, 트리플킬, 쿼드라킬, 펜타킬, 미쳐 날뛰고 있습니다, 잡았쥬, 보스 컷, 보스 클리어
- 이벤트 큐: 피격, 벽 충돌, 시점 전환, 기울기, 궁 준비 완료, 궁 발사, UI 음성

## UI 시스템

초기에는 C++ HUD Canvas 중심으로 구성했고, 이후 Unreal Editor에서 수동으로 배치하고 수정할 수 있도록 UMG 기반 UI 에셋을 추가했습니다.

관리 대상:

- Lobby UI
- Dashboard UI
- Character Select UI
- Gameplay HUD
- Pause / Volume UI
- Game Over / Leaderboard UI
- 적 HP bar
- 보스 HP bar

전투 중 보스가 등장하지 않았을 때는 보스 HP UI가 표시되지 않도록 처리했습니다.  
적 HP UI는 적 오브젝트 위에 표시되도록 구성했습니다.

## 데이터 테이블

밸런스 조정을 위해 데이터 테이블을 사용합니다.

### Player Stat Data

플레이어 기체별 값을 조정합니다.

- 이동 속도
- 좌우 이동 속도
- HP
- 기본 탄환 피해량
- 궁극기 피해량
- 사격 간격

### Enemy / Boss Stat Data

적과 보스의 기본 스탯을 조정합니다.

- 이동 속도
- HP
- 접촉 피해량
- 점수
- 크기
- 색상

### Wave Design Data

wave별 난이도와 스폰 조건을 조정합니다.

- wave 번호
- 스폰 간격
- 한 번에 스폰되는 적 수
- 기본 적 속도
- 기본 적 HP
- 기본 적 피해량
- 빠른 적 확률
- 탱커 적 확률
- 보스 wave 여부

## 맵과 공간 구성

게임플레이 맵은 우주 비행 코리더 느낌의 무한 맵으로 구성했습니다.

- 플레이어가 앞으로 이동해도 맵이 이어지는 느낌
- 뒤로 이동해도 맵 밖으로 이탈하지 않도록 경계 처리
- 네온 라인, 프레임, 바닥 패널을 통해 전투 공간 인식 가능
- 벽과 충돌하면 피해 발생
